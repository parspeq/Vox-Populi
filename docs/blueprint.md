# **App Name**: EchoSphere

## Core Features:

- Topic Creation: Allow users to create new discussion topics with relevant titles and descriptions.
- Threaded Replies: Enable users to post replies to existing topics, creating threaded conversations.
- Threaded View: Implement a clear and intuitive interface for viewing threaded conversations, showing the hierarchy of replies.
- Search Functionality: Implement a search feature to find discussions and replies of interest to the user.
- Text Formatting: Enable users to format posts with basic styling options such as bold, italics, and lists.
- Attachment Support: Enable users to add attachments to their posts
- AI Topic Suggestions: Implement a tool that suggests related topics based on the content of the current thread to increase user engagement and discovery.
- Peer to Peer Chat: Enable users to directly message each other.

## Style Guidelines:

- Primary color: Vibrant teal (#008080) for high contrast and modern feel.
- Background color: Dark gray (#222222) to support the dark mode theme.
- Accent color: Light cyan (#E0FFFF) to complement the teal and provide visual highlights.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Use clean, minimalist icons for key actions such as replying, creating topics, and searching.
- Maintain a clean and organized layout with clear visual hierarchy for easy navigation.
- Incorporate subtle animations for actions like loading new content and displaying replies.
